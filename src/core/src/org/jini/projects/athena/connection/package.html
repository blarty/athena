< !DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"> Package holding all the classes for connections and connection pools.
This includes:
  • Connection Interfaces:
    • AthenaConnection
    • SystemConnection
  • Connection pool classes:
    • ConnectionPool
    • PoolThread
  • Implementing connection classes:
    • RemoteConnectionImpl(AthenaConnection)
    • BeaJoltConnection(SystemConnection)
    • CTGConnection(SystemConnection)
    • OracleConnection(SystemConnection)
    • SQLConnection(SystemConnection)
Connection classes may be forward or backward facing, i.e. to the client or the host represented, but not both.
The connection pool is not a member of org.jini.projects.athena.resources.
