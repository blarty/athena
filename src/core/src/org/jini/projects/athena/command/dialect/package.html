Package containing the DialectEngine class as well as any implementing Dialects.
Dialects are the way in which a Call is processed, Calls, supplied with an appropriate command, will request a dialect from the DialectEngine class for the given Commands callname. A dialect will process the internal arguments, at which point the Call object will execute the invocation against the host, the dialect will then process the output of the cal accordingly and finally pass that data back to the call, which returns any data back to the client.
